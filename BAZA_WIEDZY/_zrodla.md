## Jak czytać oryginały

**Pliki .docx** — pandoc nie zainstalowany, więc:

```bash
unzip -p "PLIK.docx" word/document.xml | sed 's/<[^>]*>/ /g' | tr -s ' \n' ' '
```

**Pliki .pdf** — narzędzie Read obsługuje natywnie (do 20 stron na request, dla większych podawać `pages: "1-5"`).

**Pliki .pbix (Power BI)** — wymaga Power BI Desktop, brak narzędzia w Claude Code.

**Pliki .csv** — Read normalnie.

**Pliki Fireflies (przez MCP):**
- `fireflies_get_transcripts` — lista
- `fireflies_get_transcript {id}` — pełna transkrypcja
- `fireflies_get_summary {id}` — podsumowanie + action items
- `fireflies_search "temat"` — wyszukiwanie
