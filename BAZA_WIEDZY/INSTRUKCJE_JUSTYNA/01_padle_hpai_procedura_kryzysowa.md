# Justyna — procedura kryzysowa: padłe / choroby / HPAI

> Justyna, ta instrukcja domyka instrukcję Magdy #14. Magda **eskaluje do Ciebie**, Ty **decydujesz**. Tu masz pełną procedurę co robić przy różnych poziomach problemu zdrowotnego stada.

---

## Twoja rola w łańcuchu

```
Lekarz wet wpisuje dane → Magda zauważa próg → Magda dzwoni do Ciebie → TY DECYDUJESZ
```

- **Magda** = czujnik (zauważa, liczy %, eskaluje)
- **Ty (Justyna)** = decydent jakościowy (limit 1000 zł, wyżej → Ser)
- **Ser** = decyzje strategiczne / finansowe / prawne
- **PIWet** (Powiatowy Inspektorat Weterynaryjny) = przy HPAI / chorobach zakaźnych

---

## 🚦 Poziomy reakcji (progi z instrukcji Magdy #14)

### 🟢 Padłe 3–5% — Magda notuje, Ty dostajesz info

**Co robisz:**
- [ ] Przyjmij info do wiadomości (mail / ZPSP notatka)
- [ ] Sprawdź czy to **jednorazowe** czy **trend** (`Statystyki Hodowców` → ranking)
- [ ] Jeśli trend (3 raz z rzędu) → telefon do hodowcy + przegląd: pasza? woda? wentylacja?
- [ ] **Nie wstrzymuj uboju** — to normalny zakres

### 🟡 Padłe 5–10% — Magda dzwoni, Ty sprawdzasz osobiście

**Co robisz:**
- [ ] **Osobiście obejrzyj partię** (jeśli jeszcze w buforze / na rampie)
- [ ] Skonsultuj z **lekarzem wet** który ją oceniał — jaka przyczyna?
- [ ] Decyzja: ubój standardowy / wstrzymanie / dodatkowe badania
- [ ] **Notatka w partii** + info do **Asi** (rozliczenie / reklamacja hodowcy)
- [ ] **Flaguj hodowcę** jeśli to powtarzalne (Ser może rozważyć rozstanie)
- [ ] Limit decyzyjny: **do 1000 zł** sama, wyżej → Ser

### 🔴 Padłe > 10% LUB jakiekolwiek ZM (zakaźne) — STAN ALARMOWY

**Co robisz NATYCHMIAST (równolegle):**
- [ ] **Wstrzymaj ubój tej partii** — nie wpuszczaj do linii dopóki nie zdecydujesz
- [ ] **Zadzwoń do lekarza wet** (zakładowego) — pilna ocena
- [ ] **Zadzwoń do Sera** — informacja + decyzja finansowa/prawna
- [ ] **Zabezpiecz próbki** (procedura HACCP — wiesz którą)
- [ ] **Izoluj partię** fizycznie (osobny bufor jeśli możliwy)
- [ ] **Notatka szczegółowa** w partii + zdjęcia

## 🦠 HPAI (ptasia grypa) — procedura specjalna

**Sygnały podejrzenia HPAI:**
- Nagły masowy padłe (> 10-15% w krótkim czasie)
- Objawy neurologiczne / oddechowe opisane przez hodowcę
- Komunikaty PIWet o ogniskach w okolicy
- Kilka hodowców z tego samego regionu z nagłym wzrostem padłych

**Procedura (KRYTYCZNA — to obowiązek prawny):**

1. **NATYCHMIAST wstrzymaj przyjęcie** tej i kolejnych partii od tego hodowcy
2. **Zadzwoń do PIWet** (Powiatowy Inspektorat Weterynaryjny) — **zgłoszenie podejrzenia HPAI jest obowiązkowe**
3. **Zadzwoń do Sera** — uruchamia procedurę kryzysową firmy
4. **Lekarz wet zakładowy** — pobiera próbki, kontakt z laboratorium
5. **NIE wpuszczaj** podejrzanej partii do produkcji pod żadnym pozorem
6. **Dokumentuj wszystko** — daty, godziny, kto powiadomiony, decyzje
7. **Komunikacja zewnętrzna** — TYLKO przez Sera (media, inni hodowcy, klienci)

> **Pamiętaj:** firma straciła **31 mln PLN przy poprzednim ognisku HPAI** (audyt branżowy). Procedura nie jest formalnością — to ochrona całego zakładu.

## 🔢 Jak Magda liczy próg (żebyś wiedziała co Ci podaje)

```
% padłych = liczba padłych / liczba sztuk wstawionych × 100%

Magda podaje Ci: "Justyna, partia KOWALSKI, padłe 6.5% (1950 z 30000)"
```

Klasy które lekarz wet wpisuje (Panel Lekarza, kolumny FarmerCalc):
- **Padłe** (DeclI2) — sztuki martwe
- **CH** (DeclI3) — choroby ogólnie
- **NW** (DeclI4) — niezakaźne
- **ZM** (DeclI5) — **zakaźne (najważniejsze — każda > 0 to alarm)**
- **Łapki** — klasa B (jakość niższa, nie zdrowotny)

## 🎯 Powiązanie z systemem (przyszłość — Część 3 audytu)

Po wdrożeniu **engine alertów trendów** (Centrum Asi, sekcja F):
- System automatycznie wykryje "hodowca 3-i raz padłe > 5%"
- Alert trafi do Asi (compliance) + powinien też do Ciebie (jakość)
- **Twoja rekomendacja dla Sera:** poproś żeby alerty zdrowotne (padłe/ZM) szły **też do Ciebie**, nie tylko Asi

## 💚 Zasada żelazna jakości

> **Przy wątpliwości — wstrzymaj, nie wpuszczaj.** Lepiej stracić jedną partię niż zarazić cały zakład.
> **ZM (zakaźne) = zawsze PIWet + Ser, nigdy sama decyzja.**
> **HPAI = procedura prawna, nie opcja.**
